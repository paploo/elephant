Elephant::Cache

A simple caching API.

To use, initialize the cache in your object initialize method, then simply call cache(:lookup_key){...} with the block generating the value that should be cached. From then on, it'll use the cached value (unless you dirty the cache, of course).

If you want to access a cached value outside of where it is created, it is best to cache the value in an accessor method and then call the accessor.

Note: Outside of initialize_cache, the APIs are designed in a way that easily facilitates using any cache store, as long as it has hash-like access. Thus, one could make a mapper to Memcache or Mongo and override initialize_cache with a new implementation before calling it, and you'd be able to store anywhere.
